ZenTry – Gizlilik Politikası

ZenTry olarak kullanıcılarımızın gizliliğine önem veriyoruz. Bu Gizlilik Politikası, ZenTry uygulaması kullanılırken hangi bilgilerin toplandığını, nasıl kullanıldığını ve nasıl korunduğunu açıklar.

1. Toplanan Bilgiler

1.1 Kullanıcı Tarafından Sağlanan Bilgiler

  • Kullanıcı adı
  • E-posta adresi
  • İsteğe bağlı profil bilgileri

1.2 Doğrulama Bilgileri

ZenTry uygulamasında kullanıcı doğrulaması yalnızca e-posta adresi üzerinden yapılır. Doğrulama amacıyla kullanıcının e-posta adresine tek kullanımlık bir onay kodu gönderilir. Kullanıcı bu kodu manuel olarak uygulama içerisine girerek doğrulama işlemini tamamlar.

Uygulama SMS göndermez, almaz veya telefon numarası üzerinden doğrulama yapmaz.

1.3 Otomatik Olarak Toplanan Bilgiler

  • Uygulama içi teknik hata kayıtları
  • Cihaz türü ve işletim sistemi bilgileri (kişisel veri içermez)

2. Bilgilerin Kullanım Amacı

  • Kullanıcı hesabının oluşturulması ve yönetilmesi
  • E-posta tabanlı kullanıcı doğrulamasının sağlanması
  • Uygulama güvenliğinin artırılması
  • Teknik sorunların giderilmesi

3. Bilgilerin Paylaşımı

ZenTry, kullanıcı bilgilerini üçüncü taraflarla satmaz veya paylaşmaz. Yalnızca yasal yükümlülükler kapsamında yetkili mercilerle paylaşım yapılabilir.

4. Veri Güvenliği

Kullanıcı verileri, yetkisiz erişime karşı korunur. E-posta doğrulama kodları geçici olarak kullanılır ve doğrulama sonrası saklanmaz.

5. Kullanıcı Hakları

  • Bilgilerini görüntüleme
  • Bilgilerini güncelleme
  • Hesabını silme

6. İletişim

Gizlilik politikası ile ilgili sorularınız için bizimle iletişime geçebilirsiniz:
E-posta: support@zentry.app

ZenTry – Privacy Policy

ZenTry values user privacy. This Privacy Policy explains what information is collected, how it is used, and how it is protected when using the ZenTry application.

1. Information We Collect

1.1 Information Provided by the User

  • Username
  • Email address
  • Optional profile information

1.2 Verification Information

User verification in ZenTry is performed only via email. A one-time verification code is sent to the user’s email address. The user manually enters this code into the application to complete verification.

The application does not send or receive SMS messages and does not perform phone number verification.

1.3 Automatically Collected Information

  • Application error logs
  • Device and operating system information (non-personal)

2. Purpose of Using Information

  • Account creation and management
  • Email-based user verification
  • Improving application security
  • Resolving technical issues

3. Data Sharing

ZenTry does not sell or share user information with third parties. Data may only be shared if required by law.

4. Data Security

User data is protected against unauthorized access. Email verification codes are temporary and are not stored after verification.

5. User Rights

  • Access personal data
  • Update personal data
  • Delete account

6. Contact

If you have any questions regarding this Privacy Policy, please contact us:
Email: support@zentry.app
